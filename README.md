# 🏏 IPL AI Dashboard

## Overview

The IPL AI Dashboard is an interactive data analytics dashboard built using Streamlit, Pandas, and Plotly. It provides insights into IPL matches from 2008 to 2022 through KPIs, visualizations, and interactive filters.

## Features

* Dataset Overview
* Data Cleaning & Missing Value Analysis
* Key Performance Indicators (KPIs)
* Interactive Season Filter
* 5+ Interactive Visualizations
* Insights & Conclusions

## Dataset

IPL Matches Dataset (2008–2022)

File Used:
IPL_Matches_2008_2022.csv

## Technologies Used

* Python
* Pandas
* NumPy
* Plotly
* Streamlit

## Visualizations

1. Matches per Season
2. Toss Decision Analysis
3. Top Winning Teams
4. Matches by City
5. Player of the Match Leaders

## How to Run

Install dependencies:

python -m pip install -r requirements.txt

Run the application:

python -m streamlit run app.py
